About Cocoon

Cocoon boosts data engineering productivity in ELT by >10x with better outcomes. It uses LLMs to automatically transform (by writing SQL pipelines) messy data warehouses into well-organized ones ready for analysis. Engineers review the pipelines generated by Cocoon, instead of writing them themselves.

Our Process

Cocoon automates the following transformation tasks:
  1. Data Cleaning: Deduplication, missing values handling, typos fixing, etc.
  2. Data Integration: Join key identification, fuzzy joins/unions, entity matching, schema matching, etc.
  3. Data Modeling: Entity identification, normalization, surrogate key identification/creation, etc.
Cocoon is Open Sourced
